  <h1>Newton DEX command line interface</h1>
</div>


## Installation
```
pnpm i
```

## Requirements

You need a A v4 testnet TON wallet. You also need a toncenter testnet api key (https://toncenter.com/).

## Usage

Create a .env file (use .env.example as an example) and fill it in.


```
npx tsx src/cli.ts
```
